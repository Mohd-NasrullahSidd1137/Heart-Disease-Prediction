# ❤️ Heart Disease Prediction System

An AI-powered **Heart Disease Prediction Web Application** built with **Streamlit** and **Machine Learning**. This application predicts the likelihood of heart disease based on patient clinical information using a trained Logistic Regression model.

---

## 🌐 Live Demo

🔗 https://heart-disease-prediction-1137.streamlit.app/

---

## 📌 Project Overview

Heart Disease Prediction System is a machine learning-based healthcare application that assists in predicting whether a patient is at risk of heart disease.

The application provides an intuitive medical dashboard where users can enter clinical details and instantly receive a prediction along with the probability score and health recommendations.

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/YourUsername/Heart-Disease-Prediction.git
```

Go to project directory

```bash
cd Heart-Disease-Prediction
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
